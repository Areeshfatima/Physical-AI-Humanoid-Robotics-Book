## Image Strategy

- Each chapter must have at least one visual reference (diagram, illustration, or placeholder)
- Images should follow the naming convention: `<topic-name>_<type>.<extension>`
- For example: `architecture_diagram.svg`, `data_flow.png`
- All images must have alt text for accessibility
- Image size should not exceed 5MB
- Supported formats: PNG, JPG, GIF, SVG

## Minimum Visual Requirement

Each chapter must include at least one of the following types of visuals:

- Architecture diagrams
- Data flow diagrams
- Simulation pipeline diagrams
- VLA cognition flow diagrams
- Process diagrams
- Concept illustrations
- Screenshots demonstrating functionality
