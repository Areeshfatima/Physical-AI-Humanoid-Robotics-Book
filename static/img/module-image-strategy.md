## General Image Guidelines

### Technical Requirements
- All images must be under 5MB
- Prefer vector formats (SVG) for diagrams and charts
- Use PNG for complex images with transparency
- Use JPG for photographs
- Include alt text for all images

### Accessibility Requirements
- Alt text must describe the content and function of the image
- Color contrast must meet WCAG 2.1 AA standards
- Complex images should include extended descriptions
- All diagrams should be understandable in grayscale

### Quality Standards
- All images must load within 3 seconds on standard connections
- Diagrams should be clear and unambiguous
- Screenshots should be current with the latest UI
- Images should be consistent in style across modules
- Each chapter must have at least one visual element
